## 1. Títulos y Jerarquía (Headings)
Se usan almohadillas `#`. A más almohadillas, más pequeño el título.

# Título 1 (H1)
## Título 2 (H2)
### Título 3 (H3)
#### Título 4 (H4)
##### Título 5 (H5)
