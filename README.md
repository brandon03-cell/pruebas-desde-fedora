# 🐧 Guía Maestra de Markdown (Estilo Brandon / Ubuntu)

Este es un repositorio de pruebas para masterizar el arte de Markdown en **Fedora 43**.

---

## 1. Títulos y Jerarquía (Headings)
Se usan almohadillas `#`. A más almohadillas, más pequeño el título.

# Título 1 (H1)
## Título 2 (H2)
### Título 3 (H3)
#### Título 4 (H4)
##### Título 5 (H5)

---

## 2. Énfasis de Texto (Formatting)

* **Negrita**: se hace con `**texto**` o `__texto__`.
* *Cursiva*: se hace con `*texto*` o `_texto_`.
* ***Negrita e Itálica***: se hace con `***texto***`.
* ~~Tachado~~: se hace con `~~texto~~`.
* Texto en <sub>subíndice</sub> y texto en <sup>superíndice</sup> (Usando etiquetas HTML).

---
